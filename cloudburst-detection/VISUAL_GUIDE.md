- **Active**: Slightly darker, smaller shadow
- **Disabled**: Opacity 50%, no hover effect

### Map

- **Pan**: Click + drag
- **Zoom**: Scroll wheel or +/- buttons
- **Marker Click**: Select node
- **Background Click**: Deselect node
